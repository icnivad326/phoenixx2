HTML Basic Video Player

There are several key design and interface features I look for when researching web_mobile element tutorials.

The tutorial must meet the following criteria; 1) include a detailed explaination of the code used to create the element, 2) Must work across multiple platforms and function properly in the majority of browsers (including older versions of IE).

The objective is to keep the design and code simple. The element should work well within a responsive layout and on the majority of chip-based devices.

A series of tutorials by Adam Khoury1 supplied the framework and the video player code. The player is designed to be easily modifiable for staging a variety of events. A special thanks to Videezy.com2 for the video content. Software support for video and audio editing provided by VideoPad by NCH Software.3 Please note that this player does not currently work on an Android OS version 4.0.4 Samsung phone as of August 2018. I'm currently researching a workaround to this problem.
