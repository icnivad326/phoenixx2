Infinity Water Label and Box Designs

Mockup is the “visual script” used for the presentation of the overall visual design of the product.1 Many of the packaging and custom shaped concepts today require a 3 dimensional proof of what the finished project will look like. There are several methods for creating a virtual product proof: 1) Print the one-up press file using an inkjet printer and an adhesive backing substrate. The next step would be to mount the printout to the actual product, photograph the object, save as jpeg or pdf and send or upload file. 2) Create a 3D model, crop the one-up press file and then save the cropped image to be “mapped” or “applied” to the model. 3) The method displayed on this page involves finding an online mock-up to match the project product, and then “map” the image sections using Illustrator and Photoshop 3D tools.
