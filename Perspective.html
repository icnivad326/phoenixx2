Promotions in Perspective

The majority of pages located on this site include methods for inserting elements into a responsive layout. The challenge is to incorporate a method to display the hero image in a grid-like format.

The problem: the hero images are responsive and become too small for impact when viewing on a small screen device. Possible Solution: use an image that can be divided by an even number and then insert a table <div> using HTML and CSS.

In this experiment, using "@media screen and (max-width: 600px)" will display the images at 100%; any screen size with a width below 600px will then be converted to a stacked 4 panel image. Resize the browser window to see the effect.
